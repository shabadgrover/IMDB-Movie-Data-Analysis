# IMDB Movie Data Analysis

## 📌 Overview
This project focuses on performing Exploratory Data Analysis (EDA) on the IMDB movie dataset to uncover insights related to movie ratings, genres, and overall trends in the film industry.

## 🎯 Objectives
- Analyze distribution of movie ratings
- Identify popular genres and trends
- Understand relationships between ratings, votes, and other features
- Extract meaningful insights from movie data

## 🛠️ Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## 📊 Key Steps Performed
- Data Cleaning and Preprocessing
- Handling missing values
- Feature Analysis
- Exploratory Data Analysis (EDA)
- Data Visualization

## 📈 Key Insights
- Identified most popular and highest-rated genres
- Analyzed rating distributions across movies
- Observed trends in movie popularity and audience preferences
- Explored correlations between different features

## 📷 Visualizations
The project includes:
- Distribution plots
- Bar charts
- Heatmaps
- Genre analysis graphs

## 🚀 Outcome
This project demonstrates how movie data can be analyzed to uncover trends and patterns using data analysis and visualization techniques.

## 📌 Future Scope
- Build recommendation system based on movie preferences
- Perform sentiment analysis on reviews
- Create interactive dashboard for movie insights
